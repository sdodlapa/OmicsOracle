## Documentation Standards

- **Current docs**: Place in `docs/current/` or appropriate subdirectory
- **Historical docs**: Auto-archived monthly to `docs/archive/YYYY-MM/`
- **Naming**: Use descriptive names, avoid dates in filenames (except for archive)
- **Updates**: Keep README.md updated with new major documents
